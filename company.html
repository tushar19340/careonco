OVERVIEW

Early stage investment in a R&D focused company that has developed path breaking products for cancer care and early detection and screening/diagnosis.

UNIQUE

CareOnco is extremely unique because we are using a technique which is not relying on the tumour cells for detection.

It only uses the components of the blood that is already there (platelets) and looks for signatures which makes this extremely accurate ( very high sensitivity)

ACHIEVEMENTS

The intellectual assets that we have built is in the area a low cost testing, AI, ML and Deep Learning tools to build a robust “Bench to Bedside” assembly line for the products that we will introduce globally – an early detection gene pool to help in analyzing cancer.
